✓ Solution 5 of 5

Haversine Formula

Calculates approximate distance between two points on Earth's surface. Based on Earth's radius — very accurate compared to simpler methods.

⚠ Note: Not 100% precise — affected by terrain, non-linear roads, elevation. But more than sufficient for tracking. Widely used in GPS & IoT.

6,371 km

Earth Radius

~99.9%

Accuracy

φ = Latitude · λ = Longitude · R = 6371km · d = Distance
a = sin²(Δφ/2) + cos(φ₁)·cos(φ₂)·sin²(Δλ/2)
─────────────────────────
c = 2·atan2(√a, √(1−a))
─────────────────────────
d = R · c
haversine.cpp
// Haversine Formula
#define R 6371000

double haversine(double lat1, double lon1,
                  double lat2, double lon2) {
  double dLat = toRad(lat2 - lat1);
  double dLon = toRad(lon2 - lon1);
  double a = sin(dLat/2)*sin(dLat/2) +
    cos(toRad(lat1))*cos(toRad(lat2))*
    sin(dLon/2)*sin(dLon/2);
  double c = 2*atan2(sqrt(a),sqrt(1-a));
  return R * c; // meters
}
// ETA in minutes
int calcETA(double d, double spd) {
  if(spd<1)return -1;
  return(int)((d/1000.0)/spd*60);
}
