✓ Solution 5

Map Distance: Haversine

The Formula

a = sin²(Δφ/2) + cos(φ₁)·cos(φ₂)·sin²(Δλ/2)
──────────────────────────────
c = 2·atan2(√a, √(1−a))
──────────────────────────────
d = R · c

Based on Earth's spherical radius. Provides highly accurate real-world distances between GPS coordinates — widely used in navigation and IoT systems.

haversine.cpp
// Haversine Formula
#define R 6371000 // Earth radius (m)

double haversine(
    double lat1, double lon1,
    double lat2, double lon2
) {
    double dLat = toRad(lat2 - lat1);
    double dLon = toRad(lon2 - lon1);

    double a =
        sin(dLat/2)*sin(dLat/2) +
        cos(toRad(lat1)) *
        cos(toRad(lat2)) *
        sin(dLon/2)*sin(dLon/2);

    double c = 2*atan2(
        sqrt(a), sqrt(1-a));

    return R * c; // Result in meters
}

⚡ How It Works

Data Flow

1

Data Collection

ESP32 reads GPS coordinates and packages them with the Bus ID.

2

LoRa Transmission

Packet sent via LoRa at low frequency and power over long range.

3

Station Reception

Bus stop receives and immediately decodes the incoming packet.

4

Processing

Station processes live — no storage — then prepares for display.

5

Final Display

Web dashboard shows location, speed, and estimated arrival time.
